## Getting Started

### Prerequisites

Ensure you have Node.js (>= 20.0.0) installed. Then, install the required dependencies:

```bash
npm install
```

### Development

To start the development server:

```bash
npm run dev
```

### Build

To build the project for production:

```bash
npm run build
```

### Linting

To lint the codebase:

```bash
npm run lint
```

### Type Checking

To check TypeScript types:

```bash
npm run typecheck
```
